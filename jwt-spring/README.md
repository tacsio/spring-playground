# JWT Spring

Simple implementation of Json Web Tokens with Spring
![logo][logo]

## Steps

### Login

Use your favorite http client.

_Sample using httpie_

__Request:__
```shell script
http POST localhost:8080/login \
username=admin \
password=password
```
__Response__
```shell script
HTTP/1.1 200 
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4gRG9lIiwiaWF0IjoxNTE2MjM5MDIyfQ.SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c
...
```

### Signed Request

```shell script
http GET localhost:8080/hello \
Authorization:'Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4gRG9lIiwiaWF0IjoxNTE2MjM5MDIyfQ.SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c'
```
